# SDGNN 实现说明

## 公开代码状态
截至2026年4月，未检索到可确认的SDGNN官方代码仓库。
因此本仓库区分两条实现口径：

- `b0`：`SDGNN-compatible`，用于正文 B0~B5 消融链，方便和 MSAS-GNN 共享工程主干；
- `sdgnn_pure`：尽量贴近原始论文的候选集与 Phase-Theta 训练协议。

为保证正文实验中各分解式方法的可比性，当前仓库默认将 `b0`、`sdgnn_pure` 与 `b5`
统一挂接到同一套两层 `GCN` 教师模型上；差异仅来自候选集构造、正则化参数和交替优化协议，
而不来自教师骨干网络本身。

## 实现依据
- 论文：arXiv:2410.19723v2，Section 3
- 优化目标：min_{Θ,W} (1/2)‖H* - Θ^T·Φ̃‖_F^2 + λ·Ω(Θ)
- 推理：Ĥ = Θ^{fixed,T}·Φ̃
- 正则系数：全局统一 λ（所有节点相同）
- 训练流程：原论文 Algorithm 1 明确采用交替优化，先更新稀疏权重 `θ_z`（Phase-Θ），再更新变换参数 `W`（Phase-ϕ / Phase-W）

## 两种 SDGNN 口径
| 属性 | `b0` / SDGNN-compatible | `sdgnn_pure` |
|------|--------------------------|---------------|
| 正则系数 | 全局统一 λ | 全局统一 λ |
| 教师模型（当前默认） | 两层 GCN | 两层 GCN |
| 候选集 | 分层 BFS 环层候选集 | `K-hop` 全保留 + `D-hop` 递归 fanout 采样 |
| Phase-Θ | 分层残差级联 LARS/Lasso | 平坦候选池上的单次 LARS/Lasso |
| Phase-W | 显式 `W_phi` 交替优化 | 显式 `W_phi` 交替优化 |

## 与MSAS-GNN区别
| 属性 | SDGNN-compatible (B0) | MSAS-GNN (B5) |
|------|-----------|--------------|
| 正则系数 | 全局统一 λ | 节点级 τ(i) |
| 跳距预算 | 无分层 | k_i^(l) |
| 频率信息 | 无 | 等权谱能量 |
| 交替优化 | 有 Phase-Θ / Phase-W，但参数为全局统一 λ | 有 Phase-Θ / Phase-W，且引入节点级 τ(i) 与跳距预算 |
