# PDF-Word 严格一致核对清单

目标：将 `main-final.docx` 与 `main.pdf` 在视觉和结构上尽量对齐（答辩提交级）。

## 0. 前置检查

- [ ] `pandoc-warnings.log` 无未处理引用告警（missing citation）
- [ ] `pandoc-warnings.log` 中公式解析告警已逐条核对
- [ ] Word 文档中无明显 `\LaTeX` 原始命令残留

## 1. 页面级一致性

- [ ] 页面尺寸：A4
- [ ] 上下左右边距与 PDF 一致
- [ ] 页眉、页脚、页码位置一致
- [ ] 章节是否从新页开始（与 PDF 一致）

## 2. 文字样式一致性

- [ ] 正文字体（中文/英文）一致
- [ ] 正文字号、行距、首行缩进一致
- [ ] 标题层级（章/节/小节）字号和粗细一致
- [ ] 标题编号层级一致（章号、节号）

## 3. 目录与清单

- [ ] 目录层级、缩进、页码对齐
- [ ] 插图清单、附表清单、算法清单齐全
- [ ] 目录页码与正文页码跳转正确

## 4. 图、表、算法

- [ ] 图编号顺序与 PDF 一致
- [ ] 表编号顺序与 PDF 一致
- [ ] 图题位置在图下，表题位置在表上
- [ ] 长表跨页标题续表格式一致
- [ ] 算法环境样式（标题、输入输出）一致

## 5. 公式与交叉引用

- [ ] 公式全部可见，无乱码、无截断
- [ ] 行间公式居中、编号右对齐
- [ ] 行内公式基线和字号协调
- [ ] 公式引用（如“式(3-1)”）编号一致
- [ ] 图表引用、章节引用跳转正确

## 6. 参考文献

- [ ] 文中引用编号与 PDF 一致
- [ ] 参考文献条目数量一致
- [ ] 参考文献排序规则一致（顺序制/著者年）
- [ ] 每条文献字段完整（作者、题名、期刊、年份等）

## 7. 附录与后置部分

- [ ] 附录 A/B/C 标题和编号一致
- [ ] 索引、简历、致谢、委员会名单完整
- [ ] 各后置部分分页位置与 PDF 基本一致
