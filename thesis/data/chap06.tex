\chapter{实验评估}
\label{chap:ch6-experiment}

第3至5章分别完成了问题形式化、三维自适应机制设计与算法整合，本章将上述方法设计与分析结论转化为可量化的实验证据。实验重点考察两个方面：MSAS-GNN 在引文网络、网页图和大规模图三类场景中相对现有稀疏 GNN 方法的精度与效率表现；以及各核心模块的独立贡献，并检验第4章关于节点级正则系数设计与跳距预算分配、第5章关于谱近似误差分析方向是否与实验观察一致。主实验及正文中参与显著性检验的结果以 Cora、Citeseer、PubMed、ogbn-arxiv、Chameleon、Squirrel 六个静态基准数据集为基础，在10个随机种子下重复，报告均值$\pm$标准差，并采用 Wilcoxon 符号秩检验\cite{Wilcoxon1945Ranking}（双侧，$p<0.05$）评估显著性；个别仅保留均值记录的补充列项在相应表注中单独说明，不纳入显著性检验。具体实验设置见第~\ref{sec:ch6-setup}~节，主实验结果见第~\ref{sec:ch6-main}~节，消融与可视化分析分别见第~\ref{sec:ch6-ablation}~节与第~\ref{sec:ch6-visual}~节，效率评估见第~\ref{sec:ch6-efficiency}~节。

本章实验从精度、模块贡献与效率三个层面系统验证本文方法的有效性。主实验在六个基准数据集上比较MSAS-GNN与各基线方法的节点分类准确率，用以衡量三维自适应机制带来的整体性能增益。消融实验在受控条件下逐一拆除各模块，定量分离谱能量、度中心性、局部图熵与分层跳距预算各自的独立贡献，识别节点谱能量作为主导改进因子的地位。效率分析在真实GPU环境下测量推理时间、显存占用与预处理摊销，验证固化稀疏矩阵乘口径的复杂度收益。三类实验共同构成从方法设计到实际部署价值的完整评估链路。

\section{实验设置}
\label{sec:ch6-setup}

实验采用六个公开数据集，统计信息见表~\ref{tab:ch6-datasets}。数据划分统一为60\%/20\%/20\%随机划分，ogbn-arxiv采用OGB\cite{Hu2020OGB}官方时间划分（训练集为2017年及以前，验证集为2018年，测试集为2019年及以后）；Chameleon与Squirrel采用PyG中\texttt{geom\_gcn\_preprocess=True}的预处理版本以保持特征表示的一致性；本文不使用其自带划分mask，而在同一数据表示上重新执行60\%/20\%/20\%随机划分，以统一各方法的评测协议并避免潜在的划分偏差；训练、验证与测试节点集合互不重叠，主训练流程中的标签仅用于监督训练与最终评测。需要说明的是，该设置服务于“统一协议下的受控比较”，其结果不应与采用官方固定划分或其他专用划分协议的文献报告数字直接作一一横向比较。实验环境为NVIDIA V100 32GB（小图）/A100 40GB（ogbn-arxiv），PyTorch 2.0.1 + PyTorch Geometric 2.3.1\footnote{本文实验实现、配置文件与结果整理脚本已公开于 GitHub 仓库：\url{https://github.com/changh2001/msas-gnn-repro}。}。关键超参数搜索采用贝叶斯优化框架（Optuna TPE采样器\cite{Akiba2019Optuna,Bergstra2011TPE}），以验证集准确率为目标，每个数据集独立进行50次试验；搜索范围与各数据集默认配置见表~\ref{tab:ch6-hyperparams}。为保证依赖教师表示 $H^*$ 的分解式方法在比较中的可比性，正文实验中的 SDGNN、MSAS-GNN 以及后续消融链 B0--B5 均统一采用两层 GCN 作为教师模型；因此不同方法之间的性能差异仅归因于稀疏分解、节点级正则化与跳距预算机制，而不归因于教师骨干网络本身。正文主实验表中的 MSAS-GNN 对应 B5 配置（包含谱能量、度中心性、k-core、局部图熵与分层跳距预算），并采用完整交替优化版本 B5-full：以教师表示归一化后的 $\tilde{\Phi}^{(0)}=\ell_2\text{-row-normalize}(H^*)$ 作为热启动，在每轮外循环中依次执行 Phase $\Theta$、Phase $W$ 与 $\tilde{\Phi}$ 重算；其中显式参数 $W_\phi$ 以由 $(X,H^*)$ 闭式岭回归构造的线性变换热启动，冻结特征变体 B5-frozen 及其他退化配置仅在第~\ref{subsec:ch6-modular}~节作为消融对照报告。

所有静态实验在10个随机种子下重复，种子序列为 $\{42,\ 123,\ 456,\ 789,\ 2021,\ 2022,\ 2023,\ 2024,\ 2025,\ 2026\}$，同时控制数据集随机划分、参数初始化与邻居采样过程中的随机性，确保各次运行的差异来源可溯。方法间的显著性差异采用 Wilcoxon 符号秩检验\cite{Wilcoxon1945Ranking}（双侧，$p<0.05$），以10个种子的测试准确率序列为输入；选用 Wilcoxon 检验而非配对 $t$ 检验，在于其对准确率分布不作正态假设，小样本（$n=10$）条件下具有更稳健的第一类错误控制能力。

对于 ogbn-arxiv 等节点数 $n>5\times10^4$ 的大规模图，训练阶段将算法~\ref{alg:ch5-main}的 Phase $W$ 切换为 mini-batch 实现：每轮从全部节点中随机采样子集 $B_t\subset V$（batch\_size=1024），仅在采样节点上计算 Phase $W$ 的 MSE 目标；显式参数 $W_\phi$ 以由 $(X,H^*)$ 闭式岭回归构造的线性映射热启动，并在 mini-batch Phase $W$ 中更新。Phase $\Theta$（LARS逐节点串行执行）保持不变，每次仅读取当前 $\tilde{\Phi}$ 的对应候选集行，内存需求 $O(|C_i|\cdot d)$ 与批次规模无关。训练结束后，固定最终 $W_\phi$ 对全体节点补执行一次 Phase $\Theta$ 重对齐，输出固化的 $\Theta^{\text{fixed}}$；推理接口与全图训练版本完全一致。节点数不超过 $5\times10^4$ 的小数据集保留完整邻居并执行全图单次前向，不启用上述策略。

全部实验在高性能计算集群上运行：小数据集采用 NVIDIA V100 32GB，ogbn-arxiv 采用 NVIDIA A100 40GB；软件栈为 Ubuntu 20.04 LTS、CUDA 11.8、PyTorch 2.0.1、PyTorch Geometric 2.3.1。所有效率实验在单卡单进程环境下执行，推理加速比基于"单次全图前向时间"统计（小数据集）或"ms/batch"统计（ogbn-arxiv），两者不混用。

\section{消融实验与分析结论验证}
\label{sec:ch6-ablation}

本节通过受控消融实验定量分解各模块的独立贡献，并检验前几章核心分析结论的预测方向，以 Cora 为主验证场景。

\subsection{逐模块消融分析}
\label{subsec:ch6-modular}

为避免“稀疏度”一词在不同实现口径下产生歧义，先对表~\ref{tab:ch6-ablation}中的统计量作统一说明。本文消融表中的稀疏度列不采用固化后 $n\times n$ 稀疏权重矩阵 $\Theta^{\text{fixed}}$ 的全局零元素占比，而采用\textbf{候选集剪枝率}作为报告口径：
\begin{equation}
  \mathrm{Sparsity}
  =
  \left(
  1-\frac{\sum_{i=1}^{n}\|\theta_i\|_0}{\sum_{i=1}^{n}|C_i|}
  \right)\times 100\%,
  \label{eq:ch6-ablation-sparsity}
\end{equation}

其中 $C_i=\bigcup_{l=1}^{L}\mathcal{R}_l(i)$ 为第~5~章定义的节点 $i$ 的候选集，$\|\theta_i\|_0$ 表示该候选集中最终被保留的非零邻居数。该定义衡量的是“从候选邻域到最终稀疏支撑集”的剪枝强度，因此更适合用于比较 B1--B5 与 B2-RND 等不同节点级参数设计在统一候选池口径下的保留/裁剪差异；若改用整个 $\Theta^{\text{fixed}}$ 的全矩阵零元素比例，则数值会主要受图规模 $n$ 支配，不利于体现各消融配置之间真实的结构保留差别。

\section{本章小结}
\label{sec:ch6-summary}

本章对 MSAS-GNN 进行了系统的实验评估。在精度维度，MSAS-GNN 在六个静态基准数据集上均优于直接基线 SDGNN：三个引文网络数据集平均提升约 1.4\%，两个网页图平均提升 3.2\%，大规模 ogbn-arxiv 提升 0.86\%；对应的逐数据集对比均通过 Wilcoxon 显著性检验（$p<0.05$），而“2.3 倍”这类聚合比例仅作为结果概括报告。网页图上的提升幅度高于引文网络组，说明拓扑自适应稀疏预算在结构更复杂的数据上收益更明显。在模块分析维度，消融实验确认谱能量 $E_{\text{spectral}}(i)$ 是首个且增益最大的单模块因素，并支持第4章关于节点级正则系数设计的分析方向；跳距预算消融进一步表明，$\xi=0.5$ 相比均匀分配具有更优的精度—误差权衡。对第5章谱滤波误差界（式~\eqref{eq:ch5-tau-grace}），本文在实验上主要给出方向性支持，而不将现有结果表述为严格的定量闭合验证。在效率维度，ogbn-arxiv 上相对 GCN 的批次前向加速约为 13.6 倍，四数据集平均加速比约为 8.0 倍，显存节省 74.8\%；约 470 次全图调用即可覆盖预处理代价，说明该方法在大规模高频查询场景下具备较好的工程可行性。

本章主要完成对静态主线的实验验证；全论文层面的局限性与未来研究方向统一在第7章集中讨论，此处不再重复展开。
