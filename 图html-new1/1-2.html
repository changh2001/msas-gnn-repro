第 3 章 问题形式化与
图复杂度指标体系
输入 观测图 \(G=(V,E,\mathbf{X})\)，基准GNN输出 \(H^*\)，全局统一 \(\lambda_{\mathrm{reg}}\)（SDGNN基线）
建立统一优化目标 \(\min_{W_\phi,\Theta}\mathcal{L}_{\text{MSAS}}\)，提出三项设计准则
Lanczos谱分解 \(\mathcal{O}(mK_{\text{eig}})\)，提取谱间隙 \(\hat{\lambda}_{\text{gap}}\)
节点谱能量 \(E_{\text{spectral}}(i)=\frac{1}{K_{\text{eig}}}\sum_{k=1}^{K_{\text{eig}}}\lambda_k|U_{ik}|^2\) 逐节点刻画谱分布（等权实例化）
局部图熵 \(H(i)=-\sum_j p_{ij}\ln p_{ij}\) 量化邻域不确定性
度中心性 \(C_{\text{deg}}(i)\) 与 k-core 指标刻画拓扑重要性
四类指标由算法~3.1 离线批量预计算并缓存；频率维采用等权实例化，不引入标签统计量
输出 四类可计算指标：\(\hat{\lambda}_{\text{gap}},\;E_{\text{spectral}}(i),\;H_{\text{norm}}(i),\;C_{\text{deg}}(i),\;\mathrm{core}(i)\) → 第4章

第 4 章 谱驱动三维
自适应稀疏机制
输入 第3章四类指标 \(\{\hat{\lambda}_{\text{gap}},E_{\text{spectral}}(i),H_{\text{norm}}(i),C_{\text{deg}}(i),\mathrm{core}(i)\}\)（纯结构统计量驱动）
频率维：等权实例化 \(w_k(i)=1/K_{\text{eig}}\) → 谱能量 \(E_{\text{spectral}}(i)\)（精细感知留待后续工作）
节点维：\(E_{\text{spectral}}(i),C_{\text{deg}},\text{core},H\) 多因子融合 → 谱扰动推导节点正则系数 \(\tau(i)\)
跳距维：\(\hat{\lambda}_{\text{gap}}\)/\(p_l\) → 分层配额分配 → \(k_i^{(l)}\)（\(\mathrm{RoundQuota}\) 算子实现）
三维设计相互独立，频率维输出驱动节点维，三维共同进入第5章
输出 三维自适应参数：\(\tau(i),\;k_i^{(l)},\;w_k(i)\) → 第5章LARS主循环

第 5 章 算法整合、误差
分析与复杂度分析
输入 三维参数 \(\tau(i),k_i^{(l)},w_k(i)\) 与 \(\mathcal{R}_l(i)\)（跳距配额分布）
算法整合：LARS求解器端到端交替优化，算法5.1/5.2，输出固化参数 \(\Theta^{\text{fixed}}\)
谱保持理论：\(\sigma\)-谱相似性框架，建立稀疏化误差上界与精度保证
复杂度分析：推理主项 \(\mathcal{O}(n\bar{k}d)\)，训练 \(\mathcal{O}(n\bar{k}^2+mK_{\text{eig}})\)，逐阶段口径明确
推理阶段执行单次稀疏矩阵乘，\(\bar{k}\ll\bar{d}^L\)，在线延迟可控
输出 固化参数 \(\Theta^{\text{fixed}}\)，误差界，\(\mathcal{O}(n\bar{k}d)\) 推理器 → 第6章验证
